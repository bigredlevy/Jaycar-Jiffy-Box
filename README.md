# Jaycar-Jiffy-Box
Eagle library for PCB dimensions to suit "Jiffy Box" enclosures availiable from Jaycar Electronics

This eagle library includes PCB footprints for:
Lid - PCB replaces Jiffy Box lid. Useful for printed text on your enclosure.
Internal - PCB sits inside enclosure using the integrated holding clips.
